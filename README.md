# markdown-splitter

Split markdown to multiple markdown files by `---`(HorizontalRule).

## Installation

    npm install markdown-splitter

## Usage

```sh
$ markdown-splitter all.md --output /path/to/dir [ --summary /path/to/SUMMARY.md ]
```
